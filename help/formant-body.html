Formant Body Documentation

Overview

Resonant filtering creating vowel-like character - the “throat” of Kala. Uses two parallel resonant bandpass filters (biquad) to create formant peaks that give sounds vowel-like qualities, mimicking the resonant cavities of human vocal tracts.

Category

Shaping (Orange) - Transforms sound character

Ports

Formant System Architecture

Dual Parallel Filters

The Formant Body uses two independent bandpass filters:

F1 Filter (First Formant)

  • Frequency Range: 200-1000 Hz (typical vowel fundamentals)
  • Purpose: Creates the “body” of vowel sound
  • Character: Lower frequency resonances give warmth and fullness

F2 Filter (Second Formant)

  • Frequency Range: 500-4000 Hz (vowel character defining region)
  • Purpose: Defines vowel identity and clarity
  • Character: Higher frequencies determine vowel type

Signal Flow

  Input ──┬─→ F1 Bandpass Filter ──┐
          │                        │
          └─→ F2 Bandpass Filter ──┼─→ Blend → Mix → Output
                                   │
                                   └─→ Direct Signal

Processing Algorithm

Biquad Bandpass Filters

Each formant uses a 2nd-order bandpass filter (biquad structure):

Coefficient Calculation

double omega = 2π * frequency / sampleRate;
double sinOmega = sin(omega);
double cosOmega = cos(omega);
double alpha = sinOmega / (2.0 * Q);

double a0 = 1.0 + alpha;
b0 = alpha / a0;
b1 = 0.0;
b2 = -alpha / a0;
a1 = -2.0 * cosOmega / a0;
a2 = (1.0 - alpha) / a0;

Sample Processing (Direct Form I)

output = b0 * input + b1 * x1 + b2 * x2 - a1 * y1 - a2 * y2;

// Update filter state
x2 = x1; x1 = input;
y2 = y1; y1 = output;

Formant Blending

F1/F2 Balance

// f1f2Balance: 0.0 = F2 only, 1.0 = F1 only
formantBlend = f1Output * f1f2Balance + f2Output * (1.0 - f1f2Balance);

Direct/Filtered Mix

// directMix: 0.0 = all filtered, 1.0 = all direct
output = input * directMix + formantBlend * (1.0 - directMix);

Practical Applications

Sound Design Techniques

1. Vocal Synthesis

  • Talking Effects: Automate F1/F2 frequencies to create speech-like movement
  • Choir Effects: Use multiple Formant Bodies with slightly different settings
  • Vowel Morphing: Crossfade between formant presets

2. Instrument Character

  • Acoustic Simulation: Match formant frequencies to real instruments
  • Hybrid Instruments: Combine acoustic formants with synthetic sources
  • Enhanced Realism: Add subtle formant shaping to any sound

3. Creative Effects

  • Formant Sweeps: LFO on F2 frequency for “wah-wah” effects
  • Resonance Modulation: Dynamic Q changes for expressive filtering
  • Formant Rhythms: Gate formant control for rhythmic texture

Performance Parameters

Formant Frequency Selection

  • Male Voice: F1 300-700 Hz, F2 800-2000 Hz
  • Female Voice: F1 400-900 Hz, F2 1200-3000 Hz
  • Child Voice: F1 500-1000 Hz, F2 1500-3500 Hz
  • Low Instruments: Lower F1, moderate F2
  • High Instruments: Higher F1 and F2

Q Factor Effects

  • Low Q (1-3): Broad, subtle formants, natural sound
  • Medium Q (4-10): Defined formants, balanced character
  • High Q (11-20): Sharp, resonant peaks, pronounced character

Direct Mix Control

  • 0.0 - 0.2: Strong formant character, highly filtered
  • 0.3 - 0.7: Balanced direct and formant content
  • 0.8 - 1.0: Minimal formant effect, mostly direct signal

Troubleshooting

No Formant Effect

  1. Verify signalIn is connected from Spectrum to Signal
  2. Check F1 and F2 frequencies are in audible range
  3. Ensure directMix < 1.0 (allows filtered signal through)
  4. Verify f1f2Balance isn’t at extremes (0 or 1)

Harsh/Thin Sound

  1. Increase F1 frequency for more body
  2. Adjust f1f2Balance toward center (0.5-0.7)
  3. Increase Q factors for stronger resonance
  4. Reduce directMix to emphasize formants

Muffled Sound

  1. Decrease F1 frequency for clarity
  2. Increase F2 frequency for brightness
  3. Reduce Q factors for broader resonances
  4. Increase directMix for more direct signal

Self-Oscillation/Instability

  1. Check Q factors aren’t excessively high (>15)
  2. Verify frequencies aren’t too low for sample rate
  3. Ensure input signal isn’t too loud
  4. Reduce sample rate or formant gains if needed

Unwanted Resonance

  1. Lower Q factors to reduce peak sharpness
  2. Adjust formant frequencies away from problem ranges
  3. Increase directMix to dilute formant effect
  4. Use multiple Formant Bodies for smoother transitions

Best Practices

Parameter Optimization

  1. Start with vowels: Use known vowel formant frequencies as starting points
  2. Balance formants: Adjust f1f2Balance for desired character
  3. Moderate Q values: 4-12 typically sounds most natural
  4. Controlled direct mix: 0.2-0.5 often works best

Signal Chain Design

  1. Position after Spectrum: Formant Body needs audio-rate input
  2. Allow sufficient harmonics: More harmonics = richer formant response
  3. Consider gain staging: May need level adjustment after formants
  4. Multiple formants: Chain multiple Formant Bodies for complex vowels

Modulation Strategies

  1. Slow frequency changes: 0.1-2 Hz for expressive vowel morphing
  2. Q automation: Dynamic resonance for expressive filtering
  3. Balance modulation: Crossfade between different vowel characters
  4. Direct mix control: Dynamic filtering vs dry signal

Performance Considerations

  1. Filter stability: Extreme Q can cause instability
  2. Sample rate limits: Very high frequencies may alias
  3. CPU efficiency: Formant Body is lightweight, can use many instances
  4. Real-time safety: Avoid extreme parameter changes during audio

Technical Notes

Filter Mathematics

Bandpass Transfer Function

H(s) = (s²/Q + 1/Q) / (s² + s/Q + 1)

Where s = jω, ω = 2πf, Q = quality factor

Frequency Response Characteristics

  • Center Frequency: Peak response at f1Freq/f2Freq
  • Bandwidth: f_center / Q (higher Q = narrower band)
  • Insertion Loss: 0 dB at peak (constant gain implementation)

Biquad Advantages

  • Computational Efficiency: 5 multiplications, 4 additions per sample
  • Numerical Stability: Direct Form I implementation
  • Memory Efficient: Only 4 state variables per filter

Formant Acoustics

Vocal Tract Model

  • F1: Represents pharynx cavity resonance
  • F2: Represents oral cavity resonance
  • Combined Shape: Creates vowel-specific spectral envelope
  • Natural Variations: Real voices have additional formants (F3, F4)

Parameter Relationships

  • Vowel Space: Different F1/F2 combinations create vowel continuum
  • Speaker Variations: Size and shape affect optimal frequencies
  • Dynamic Range: Real voices continuously vary formants during speech
