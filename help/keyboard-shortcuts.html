Keyboard Shortcuts

A complete reference for all keyboard shortcuts in Kala. Shortcuts marked as global work from any window without needing to switch focus first.

Score Canvas (Composition)

Tools

ShortcutAction

Track Selection

Tracks are shown as colored vertical bars on the left of the Score Canvas. Only notes belonging to selected tracks can be clicked or lasso-selected.

ShortcutAction
Click track barSelect that track only — deselects all others
Ctrl+Click track barAdd track to the selection (or remove it if already selected). At least one track is always selected.
Right-click track barContext menu — Delete Track

When multiple tracks are selected, note clicks and rubber-band lasso work across all selected tracks simultaneously. The active track (solid white border) is the one that receives newly drawn notes.

Segment Editing Mode

Active after double-clicking a quantized continuous note.

ShortcutAction
Left / RightSelect previous / next segment
DeleteDelete selected segment
