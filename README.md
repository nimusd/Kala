Kala is a compositional tool that breaks from keyboard thinking. Using a pen tablet as primary input, it captures expressive gesture as the notation itself — the act of composing IS the performance. Voices share a sonic DNA, built on a tuning-agnostic foundation. The sound engine uses a hybrid approach with "physics modeling" (adding invented laws governing mass and inertia), physics modeling, wave table synthesizing and parameter coupling building a graph with various "containers" that affect the sound like envelopes, LFOs, filters, formants etc. — creating sounds that are alive but controllable. No MIDI. No piano roll. An entirely new approach for writing and generating expressive music.
